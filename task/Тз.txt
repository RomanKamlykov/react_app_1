Необходимо сверстать главную страницу магазина с адаптивом.
Требования по реализации
По своему усмотрению использовать эффекты css (Наведение на блоки, кнопки и т.п.).
1.Первый блок - слайдер. Перелистывание должно быть вертикальным
2.Второй блок - Фильтры. Описание начиная с десктопной версии:
В шапке 7 элментов меню. Каждый из пунктов фильтрует свои товары, all содержит все элементы. 
Если выбранный пункт содержит больее 8 элементов то появляются стрелочки перелистывания (правый верхний угол блока)
Планшетная версия - все то же самое что и декстоп, за исключением того что товаров в блоке отображается по 9
Мобильная версия - Каждый из табов перестраивается в слайдер (Видно по заголовкам слайдеров). 
По умолчанию видно 3 слайдера, под ними есть кнопка next со стрелками, 
по клику идет перелистывание к следующим 3м слайдерам (Категорий всего 6, не включая all).
3.Обычный слайдер.
